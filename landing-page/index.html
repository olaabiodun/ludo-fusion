1. Overview & Scope

This Privacy Policy outlines how Ludo Fusion ("we", "us", "our"), developer of the mobile application Ludo Fusion (Package ID: com.ludofusion), processes personal user profiles and operational system logs. By engaging with our servers, you acknowledge the secure handling methods described herein.

2. Data We Safely Collect

To enable safe and robust multiplayer matchmaking and profile sync persistence, we collect specific parameters:

  • User Account Identities: Authentic user emails, screen-names, and profile avatars securely bound via Supabase Auth services.
  • Database Profiles & Match Ledger: Logs of competitive coin distribution, XP growth, and game state signatures generated by secure atomic Postgres routines.
  • Device Telemetry: Device hardware models, OS build numbers, and installation identifiers utilized strictly for fraud mitigation and server diagnostics.

3. Core Operational Data Usage

Data collected is strictly operational. We do not participate in third-party data broker sales. Information is used to:

  • Maintain low-latency server-side matchmaking boards.
  • Protect matchmaking games from external cheating software or balance tampering.
  • Enforce structural database lock mechanisms to avoid transaction concurrency conflicts.

4. Player Data Deletion Rights

In accordance with Google Play developer policies, players hold absolute rights to request complete data and account erasure. Initiating a deletion request immediately tags your profile, matchmaking statistics, and game state balances for total, irreversible deletion. This can be processed directly via our Data Deletion panel on this website, or through your in-game profile configuration menu.

5. Deep Technical Protection

Our server cluster stores data utilizing robust end-to-end database encryption. Safe Postgres RPC calls handle data allocations strictly server-side, preventing front-end injection. We consistently verify and update security certificates to ensure a locked network interface.
